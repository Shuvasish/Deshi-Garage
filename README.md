# Dashi Garage

[Live link](https://deshi-garage.web.app/)

## Features

- User can order our services

- User can payment via stripe

- User can give feedback

- User can watch current condition of their order

- Admin can add, delete, update services

## Here i used

- react

- react-router-dom

- framer motion

- styled components

- axios

- imgbb
